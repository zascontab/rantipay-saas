## Develop your own services

The creation of new services is powered by [kratos layout](https://go-kratos.dev/en/docs/getting-started/start/#project-layout)

```shell
kratos new <name> -r https://github.com/go-saas/kit-layout.git
```

For more information, go to [create your own module](../modules/create-your-own)
